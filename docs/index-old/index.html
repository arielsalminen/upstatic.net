A better way to deploy static websites to Amazon S3. Works with hand coded websites in addition to Jekyll, Middleman or any other static site generator.

p.s. It’s free! Download or Install via pgier

Feature highlights

  • Blazingly fast! All your files are Gzipped, cached and distributed globally using Amazon’s CDN.
  • Automatically creates S3 bucket and configures it for website hosting.
  • Only changed files are re-deployed, which allows faster performance.
  • No need to use clunky FTP clients anymore.
  • Concurrent file uploads.
  • Written in Go.
